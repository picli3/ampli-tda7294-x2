# ampli-tda7294-x2

Amplificador de audio basado en **dos TDA7294**, pensado como una solución práctica para construir una etapa estéreo de potencia usando un integrado muy conocido en audio DIY.

## Propósito del proyecto
Este repositorio reúne el diseño electrónico y mecánico de una placa orientada a implementar un amplificador con **dos canales**, usando el TDA7294 como núcleo de cada etapa.

La idea principal del proyecto es ofrecer una base para:
- construir un amplificador estéreo
- reutilizar una topología conocida y fácil de conseguir
- integrar PCB, esquemático y soporte mecánico/disipación
- facilitar prototipado o fabricación de una etapa de audio casera o semiprofesional

## Qué incluye
- esquemáticos del amplificador
- diseño PCB en KiCad
- documentación exportada en PDF/HTML
- archivos mecánicos del disipador o soporte en FreeCAD

## Enfoque técnico
El TDA7294 es un integrado muy usado en amplificadores de audio por su simplicidad de implementación y buena potencia para aplicaciones domésticas, educativas o DIY. En este proyecto se usa por duplicado, lo que permite montar una etapa de dos canales sobre una misma base de diseño.

## Aplicaciones típicas
- amplificador estéreo DIY
- sistema de audio casero
- etapa de potencia para pruebas de laboratorio
- proyecto educativo de audio analógico
